- 스터디를 개설하고 본격적으로 프로젝트를 진행하기 앞서 워밍업을 목적으로 시작한 프로젝트입니다.

## 🎬 프로젝트 실행 방법

1. 먼저, 깃허브 저장소를 클론하십시오.

```
git clone <저장소 URL>
```

2. 저장소 폴더로 이동합니다.

```
cd <저장소 폴더>
```

3. 필요한 의존성을 설치하십시오.

```
yarn install
```

4. 프로젝트를 실행하십시오.

```
yarn dev
```

5. 웹 브라우저에서 `http://localhost:5173/crud`에 접속하여 프로젝트를 확인합니다.
